# Just Did It Fitness Tracker v8 Exercise Flow

This version cleans the app back down to the main job: track one exercise, save it, then move to the next exercise.

## What changed
- Restored a simple app-style version of the approved Just Did It basketball logo direction
- Removed the cluttered workout-builder layout
- Log one exercise at a time
- Track each set individually
- Track reps and weight for every set
- Save the exercise before moving to the next one
- View today’s exercises
- View history
- Repeat a previous exercise entry
- Export/import backup data

## Volume formula
reps × weight for each set, added together for the exercise total.

The app keeps kg as the default because this is the metric repo, but lb is available from the unit selector.
